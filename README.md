# Astrologer Monorepo

A modern, high-precision astrology library for Node.js and TypeScript.

## Packages

- **[@astrologer/astro-core](./packages/core)**: Calculation engine (Ephemeris, Planets, Houses, Aspects).
- **[@astrologer/charts](./packages/charts)**: SVG rendering engine (Natal, Transit, Synastry, Astrotheme styles).

## Quick Start

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```
